Project Euler Problem 3: Largest prime factor

Problem:

The prime factors of 13195 are 5, 7, 13 and 29.

What is the largest prime factor of the number 600851475143 ?

Solution:

Về cơ bản, đây là bài toán phân tích một số thành thừa số nguyên tố và chọn ra số nguyên tố lớn nhất.

Để phân tích một số N thành thừa số nguyên tố:

  • Ta sẽ liên tục chia N cho số nguyên tố t(k) cho tới khi không thể tiếp tục được nữa. Tới đây:
  • Nếu N > t(k) thì ta tìm số nguyên tố t(k+1) quay lại bước trên.
  • Nếu N < t(k) thì t(k) là số nguyên tố lớn nhất. Mã code với Rust.:
fn get_next_prime_number(arr: &Vec<i64>) -> i64 {
    let last_prime = arr.last().unwrap();
    let mut next_prime = last_prime + 1;
    loop {
        let mut is_valid = true;
        for x in arr {
            if next_prime % x == 0 {
                is_valid = false;
                break;
            }
        }
        if is_valid {
            break;
        } else {
            next_prime = next_prime + 1;
        }
    }
 
    return next_prime;
}
 
fn largest_prime_factor(mut n: i64) -> i64 {
    let mut arr = Vec::new();
    arr.push(2);
    let mut current_prime: i64;
    loop {
        current_prime = arr[arr.len() -1];
        while n % current_prime == 0 && n >= current_prime {
            n = n / current_prime;
        }
 
        if n > current_prime {
            let temp = get_next_prime_number(&arr);
            arr.push(temp);
        } else {
            break;
        }
    }
    return current_prime;
}
 
fn main() {
    println!("Larngest prime factor is: {}", largest_prime_factor(600851475143));
}
 
#[test]
fn test_get_next_prime() {
    let mut arr = Vec::new();
    arr.push(2);
    let mut i = 0;
    while i < 10 {
        let temp = get_next_prime_number(&arr);
        arr.push(temp);
 
        i = i + 1;
    }
    assert_eq!(arr.len(), 11);
    assert_eq!(arr[1], 3);
}
 
#[test]
fn test_largest_prime_factor() {
    assert_eq!(largest_prime_factor(13195), 29);
}
